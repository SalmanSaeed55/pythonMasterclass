# pythonMasterclass

A repository containing material learnt from the Python Programming Masterclass course on Udemy by instructor Tim Buchalka. I have embarked on this course to understand Python syntax in depth, and to strengthen my understanding of programming, how to solve problems through code, and the overall power of Python

## Repo Structure

The repository is organized into 13 comprehensive sections, each covering progressively advanced Python programming concepts—from foundational syntax to advanced algorithms and database management. Below is a detailed breakdown of each module:

| # | Directory | Topic | Description |
|---|-----------|-------|-------------|
| 1 | **03_Fundamentals** | Python Basics | Introduction to Python, setting up the environment, and writing your first programs |
| 2 | **04_Program_Flow_Techniques** | Control Flow | Control flow techniques, including loops and conditionals |
| 3 | **05_Lists_and_Tuples** | Data Structures | Working with lists and tuples, fundamental data structures in Python |
| 4 | **06_Functions_Intro** | Functions | Introduction to functions, defining and using them, docstrings and function argument types |
| 5 | **07_Dictionaries_and_Sets** | Advanced Data Structures | Understanding dictionaries and sets, powerful data structures for storing and manipulating data |
| 6 | **08_Inputs_and_Outputs** | File I/O | Reading, writing and manipulating data to and from files, including the `csv` module |
| 7 | **09_Modules_and_Advanced_Functions** | Modules & Functions | Exploring modules and advanced function concepts, nested functions and module imports |
| 8 | **10_Date_Time_Handling** | Date & Time | Working with date and time in Python, including the `datetime` module |
| 9 | **11_GUIs_with_Tkinter** | GUI Development | Introduction to creating graphical user interfaces (GUIs) using the Tkinter library |
| 10 | **12_OOP_in_Python** | Object-Oriented Programming | OOP concepts in Python, including classes, objects and inheritance |
| 11 | **13_DB_with_SQLite3** | Database Management | Working with databases using the SQLite3 module, creating and managing databases |
| 12 | **14_Generators_and_Lambda** | Advanced Functions | Understanding generators and lambda functions, powerful tools for iterators and anonymous functions |
| 13 | **15_Algorithm_Efficiency** | Algorithms | Exploring algorithm efficiency and Big O notation, bubble sort implementation |

### Learning Path

This curriculum follows a **logical progression** from basic to advanced:
- **Fundamentals (Modules 1-3)**: Core syntax and data structures
- **Functions & Organization (Modules 4-7)**: Code organisation and modularity
- **Advanced Concepts (Modules 8-13)**: Real-world applications, GUIs, databases, and optimization

### Prerequisites

- Python 3.7 or higher
- Basic understanding of programming concepts (recommended for quick progress)
